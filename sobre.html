Perguntas sobre o site:

1. O que é um site?
É um conjunto de páginas e arquivos na internet acessíveis por navegadores.

2. O que é hospedagem?
É o serviço que armazena os arquivos do site em um servidor conectado à internet.

3. O que é um domínio?
É o endereço (ex: google.com) que facilita o acesso ao site, evitando números de IP.

4. O que é o GitHub Pages?
É um serviço gratuito do GitHub para hospedar sites estáticos (HTML, CSS e JS).

5. Qual a função do arquivo index.html?
Ele funciona como a página principal que abre automaticamente ao acessar o site.

6. O GitHub Pages suporta PHP e MySQL?
Não, ele não aceita linguagens de servidor ou bancos de dados.

7. Qual a diferença entre domínio e hospedagem?
Domínio é o endereço do site, e hospedagem é onde os arquivos ficam guardados.

8. O que acontece se o servidor ficar desligado?
O site fica fora do ar e ninguém consegue acessá-lo.
